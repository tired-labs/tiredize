# GFM Parity: HTML Blocks

## Summary

Implement HTML block support. GFM spec section 4.6 defines HTML
blocks as sequences of lines starting with specific HTML tags or
patterns. HTML blocks are not processed as markdown — their content
is passed through as raw HTML.

The parser currently has no element type for HTML blocks. Raw HTML
is treated as plain text content.

## GFM Spec Reference

Section 4.6 — HTML blocks
