# GFM Parity: Thematic Breaks

## Summary

Implement thematic break (horizontal rule) support. GFM spec
section 4.1 defines thematic breaks as lines containing three or
more `-`, `*`, or `_` characters (optionally with spaces between
them), with no other content. Examples: `---`, `***`, `___`,
`- - -`.

The parser currently has no element type for thematic breaks.
These lines are treated as plain text content within sections.
