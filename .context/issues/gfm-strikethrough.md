# GFM Parity: Strikethrough

## Summary

Implement strikethrough support. The GFM strikethrough extension
defines strikethrough as text enclosed in double tildes
(`~~text~~`). This is a GFM extension not present in the
CommonMark spec.

The parser currently has no element type for strikethrough.
Double tildes are treated as literal text.
