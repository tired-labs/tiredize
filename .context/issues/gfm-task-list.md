# GFM Parity: Task List Items

## Summary

Implement task list item support. The GFM task list extension defines
task list items as list items beginning with `[ ]` (unchecked) or
`[x]` (checked). Examples:

```
- [ ] unchecked task
- [x] completed task
```

This is a GFM extension not present in the CommonMark spec. Task
list items require list support (currently a stub — see
`markdown-type-list.md`).

The parser currently has no element type for task list items.
