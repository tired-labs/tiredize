# GFM Parity: Emphasis and Strong Emphasis

## Summary

Implement emphasis and strong emphasis support. GFM spec section 6.2
defines emphasis as text enclosed in single delimiters (`*text*` or
`_text_`) and strong emphasis as text enclosed in double delimiters
(`**text**` or `__text__`). The delimiter rules involve complex
left-flanking and right-flanking logic.

The parser currently has no element type for emphasis or strong
emphasis. Asterisks and underscores are treated as literal text.

## GFM Spec Reference

Section 6.2 — Emphasis and strong emphasis
